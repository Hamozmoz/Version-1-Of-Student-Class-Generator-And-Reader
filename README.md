What This Program Does :
This Program Allows You To
# 1 - Create A Class Of Students With By Entering :
1- A Name For The Class
2-The Number Of Students In The Class
Note : The Data Is Stored In A .md File , A Class Object Isn't Created
# 2 - Read The Student Data From The .md File , You Can Either :
1 -  Read The Data Of All Student Classes
2 - Read The Data Of A Specific Class Of Students
